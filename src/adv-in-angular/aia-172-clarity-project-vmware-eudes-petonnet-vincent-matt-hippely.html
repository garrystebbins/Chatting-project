Matt is a user interface engineer and has been working on The Clarity Project for a year. Matt has front experience as well as in many other aspects of the Clarity project. Eudes is a Web UI engineer, and well all piece of the stack in the project. Clarity is an Angular component library and full design system. The panel and guest discuss the build of the Clarity project, bootstrap, VMware, and much more. In particular, we dive pretty deep on:

  • Data Grid
  • VMware - open source

  • Innovating in Angular since beta
  • The spread across VMware before going open-source
  • Building a competent Library for Angular
  • Building another bootstrap
  • Moving the framework
  • Trust the enforcing of the buttons
  • How did package up a 3rd party library? How is you tackle it?
